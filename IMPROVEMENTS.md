# Portfolio Improvement Ideas

## Quick Wins

- [ ] **Profile photo** — Replace with a professionally cropped image on a transparent or dark background (removes the beige circle)
- [ ] **Photo centering** — The photo card vertical centering still needs fine-tuning due to BentoCard's inner wrapper div structure
- [ ] **Light mode polish** — Current design is dark-mode focused; light mode cards and glows may look washed out
- [ ] **Footer** — Currently just copyright text; add quick nav links, social icons, or a "Back to top" button
- [ ] **Section headings** — Add subtle descriptions below each heading (e.g., "Technologies I work with" under Tech Stack)

## Design Enhancements

- [ ] **Scroll progress indicator** — Thin accent-brand bar at the top of the viewport showing scroll progress
- [ ] **Active nav highlight** — Highlight the current section in the header/mobile nav as user scrolls
- [ ] **Project thumbnails** — Add screenshot images to project cards (currently text-only)
- [ ] **Project card hover** — Add a subtle image preview or gradient shift on hover
- [ ] **Smooth scroll-to-section** — Add scroll-based parallax or staggered reveal for cards entering viewport
- [ ] **Certification badges** — Replace emoji icons with actual issuer logos (Google, DevFest, etc.)
- [ ] **Experience timeline** — Add a connecting vertical line with animated dots as you scroll past entries
- [ ] **Hero card CTA buttons** — Add subtle icon animations (arrow bounce, download pulse)

## New Sections

- [ ] **Testimonials / Recommendations** — Quotes from mentors, supervisors, or collaborators
- [ ] **Blog / Writing** — Link to articles, dev.to posts, or a built-in MDX blog
- [ ] **GitHub activity** — Embed contribution graph or show recent commit activity
- [ ] **Stats bar** — Show metrics like "3 projects", "15+ technologies", "1 year experience" in a compact row

## Technical / SEO

- [ ] **Structured data (JSON-LD)** — Add Person schema for better search engine visibility
- [ ] **Open Graph image** — Generate a custom OG image for social sharing
- [ ] **Performance** — Lazy load below-fold sections, optimize image formats (WebP/AVIF)
- [ ] **Skip to content** — Add an accessibility skip link for keyboard users
- [ ] **Reduce motion** — Ensure all animations (blob drift, photo glow, typing) respect `prefers-reduced-motion`
- [ ] **PWA support** — Add manifest.json and service worker for installability
- [ ] **Analytics** — Add Vercel Analytics or Plausible for visitor tracking

## Content

- [ ] **Expand experience** — Add more detail to the internship description (bullet points of achievements)
- [ ] **Add more projects** — Showcase side projects, hackathon entries, or class assignments
- [ ] **Skills proficiency** — Add skill level indicators (beginner/intermediate/advanced) or years of experience
- [ ] **Resume** — Consider an interactive resume page instead of just a PDF download
